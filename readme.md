# C411 → ruTorrent

Extension Firefox permettant d’envoyer rapidement des torrents depuis C411 vers ruTorrent.

## Fonctionnalités
- Ajout en 1 clic avec label par défaut
- Menu clic droit pour choisir un label
- Support de l’authentification Digest
- Gestion du rate limit
- Interface de configuration simple

## Sécurité
- Les identifiants ruTorrent sont stockés localement
- Aucune donnée n’est envoyée à un serveur tiers
- Code open-source

## Installation
Disponible sur le store Firefox ou en installation manuelle.

## Configuration
1. Ouvrir l’extension
2. Renseigner l’URL ruTorrent
3. Ajouter login / mot de passe
4. Définir les labels
5. Sauvegarder
6. Tester la connexion (optionnel)
